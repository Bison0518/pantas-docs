## 关于本指南

本指南面向 **Pantas Launcher** 的所有用户，无论你是第一次使用游戏模拟前端，还是已经有过 ESDE、Pegasus等系统使用经验的玩家，都可以从中找到所需的操作说明。

**Pantas Launcher 是什么？**

Pantas Launcher 是一款运行在 Android或ios上的游戏前端应用，将你的手机（配合 Pantas Retro Pad 手柄）变成一台复古游戏机。它提供统一的游戏库界面，整合本地模拟器游戏、串流游戏、云游戏，并内置掌机助手用于手柄配置与管理。

**本指南不涉及：**
- 游戏 ROM 的获取（请遵守所在地区的相关法律法规）
- 各模拟器的详细设置（请参考对应模拟器官方文档）
- iOS 版本功能（另行提供）

## 快速上手

> 如果你刚拿到 Pantas Retro Pad 手柄，想尽快开始复古游戏，跟着这5步走。

**第 1 步：安装 Pantas Launcher**

从 Google Play Store 搜索 **"Pantas Launcher"** 并安装，或扫描包装盒上的二维码直接下载。
 
**第 2 步：安装 Pantas TF游戏卡**

将TF游戏卡插入Pantas Retro Pad的右侧内存卡槽。

**第 3 步：插入手柄**

将手机插入 Pantas Retro Pad 的活动 Type-C 接口，手柄自动开机并完成连接。  
连接成功后，出现弹窗显示打开“Pantas”来处理Gamepad吗？点击确定  
可选连接Gamepad后一律打开Pantas，实现插入手柄后自动弹出Pantas launcher。更快进入游戏！

**第 4 步：完成初始引导**

首次启动 Pantas Launcher 会进入初始配置向导：
1. 授予存储权限
2. 确认 App 配置目录（默认无需修改）
3. 选择或新建 ROM 文件夹
4. 等待自动扫描现有 ROM

**第 5 步：添加游戏**

可自动扫描或手动按**X 键**主动扫描  
等待Pantas Launcher 执行扫描完成后刷新即可显示。

**第 6 步：开始游戏**

在游戏库的模拟器板块中找到想玩的游戏，按 **A 键**启动，尽情享受吧！

## 安装与初始配置

### 权限申请

首次启动时，Pantas Launcher 需要申请以下权限，**请全部同意**，否则游戏库功能将无法正常使用：

| 权限 | 用途 |
|------|------|
| 存储空间读写 | 读取 ROM 文件及游戏封面缓存 |
| 管理所有文件 | 跨目录读取 ROM，避免权限沙箱限制 |
| 网络访问 | 游戏封面抓取（Scraper）、串流、云游戏、固件升级 |
| 通知 | 后台任务（扫描、下载）进度提示 |

> ⚠️ 如果误拒了某项权限，可前往手机【设置 → 应用 → Pantas Launcher → 权限】手动开启。

### 配置目录

Pantas Launcher 的配置文件默认存放于：

```
/storage/emulated/0/Pantas/
```

该目录结构如下：

```
Pantas/
├── gamelists/          # 游戏列表缓存（含收藏、游戏时长等）
├── themes/             # 主题文件
├── scraped/            # 封面图、截图、视频缩略图
└── configs/            # App 偏好设置
```

> 💡 建议在此目录所在的存储上留有至少 **2GB 空余空间**，用于封面图缓存。

### ROM 目录设置

ROM 文件夹可以存放在手机内部存储或 TF（MicroSD）卡中。初始引导会提示你选择：

- **使用推荐目录**：自动在内部存储创建 `/Pantas/roms/` 及各平台子文件夹
- **自定义目录**：手动选择你已有的 ROM 文件夹
- **使用 TF 卡**：选择 TF 卡根目录或其子目录

各平台 ROM 应放置在对应子文件夹中，例如：

```
roms/
├── nes/
├── snes/
├── gba/
├── nds/
├── ps1/
├── ps2/
├── psp/
├── saturn/
├── wii/
├── ngc/
└── ...
```

> 💡 如果你已有整理好的 ROM 文件夹结构，直接选择上级目录即可，Pantas Launcher 会自动识别各平台文件夹。

### 首次扫描游戏

配置完成后，Pantas Launcher 会自动扫描选定目录下的所有 ROM 文件，并按平台分类显示到游戏库中。

- **已有 ROM 的平台**会显示在游戏库的模拟器模块右下角显示游戏数量
- **没有 ROM 的平台**默认模拟器模块右下角显示游戏数量为0
- 扫描过程在后台进行，不影响正常使用

## 主界面与导航

### 界面布局

Pantas Launcher 主界面分为以下几个区域：

```
┌─────────────────────────────────────────┐
│  顶部导航栏：游戏库 / 掌机助手 / 个人中心  │  ← L1/R1 切换
├─────────────────────────────────────────┤
│                                         │
│   平台图标区  ←→  平台游戏列表           │
│   (左栏)         (右栏/主区域)           │
│                                         │
├─────────────────────────────────────────┤
│  底部：当前游戏信息 / 操作提示栏           │
└─────────────────────────────────────────┘
```

### 导航方式

Pantas Launcher 完全支持手柄操控，也支持触屏。

**手柄操作（主界面）：**

| 按键 | 功能 |
|------|------|
| 左摇杆 / 十字键 | 移动光标、切换游戏 |
| A | 确认 / 启动游戏 |
| B | 返回 / 取消 |
| X | 收藏游戏 / 扫描刷新（模拟器主界面） |
| Y | 筛选游戏 / 随机游戏（长按） |
| L1 / R1 | 切换顶部菜单页（游戏库 / 掌机助手 / 个人中心） |
| Select（短按） | 当前界面设置 / 功能菜单 |
| Start（短按） | 搜索游戏（游戏列表界面） |
| Logo 键（长按） | 呼出 / 最小化 Pantas Launcher 前端 |
| Logo 键（短按） | 呼出 AI 语音助手 |

**触屏操作：**

| 手势 | 功能 |
|------|------|
| 轻触 | 选中 |
| 双击 | 启动游戏 |
| 左右滑动 | 平台切换 |
| 长按游戏 | 弹出游戏选项菜单 |

### 手柄热键速查

以下热键在**游戏运行中**有效（需配合 RetroArch 或对应模拟器）：

| 按键组合 | 功能 |
|----------|------|
| `Menu + B` | 呼出模拟器菜单 |
| `L3 + R3` | 呼出模拟器菜单（备用） |
| `Menu + Start` | 退出游戏，返回 Pantas Launcher |
| `Menu + L1` | 读取存档 |
| `Menu + R1` | 保存存档 |
| `Menu + ← →` | 切换存档位 |
| `Menu + L2` | 游戏加速 |
| `Menu + R2` | 显示帧率 |
| `音量 +/-` | 调节音量 |

> 💡 PSP 游戏（PPSSPP）和 NDS 游戏（DraStic）的热键与 RetroArch 略有不同，详见[附录](#附录支持的模拟器与-rom-格式)。

## 游戏库

### 支持的游戏类型

Pantas Launcher 游戏库支持以下几类游戏来源：

- **模拟器游戏**：存放在 ROM 目录中的本地 ROM 文件
- **本地游戏**：已安装在手机上的 Android 原生游戏
- **串流游戏**：通过 Moonlight 等串流客户端访问 PC/主机游戏
- **云游戏**：通过网络访问云端游戏服务
- **媒体库**（v4.0 功能，敬请期待）
- **PC 游戏**（v4.0 功能，敬请期待）

### 支持的模拟器平台

以下平台在游戏库中**始终显示**（即使当前无 ROM）：

| 平台 | 模拟器 | ROM 格式 |
|------|--------|----------|
| PS2 | AetherSX2 / NetherSX2 | .iso / .bin |
| NES | RetroArch (FCEUmm) | .nes / .zip |
| SNES | RetroArch (Snes9x) | .smc / .sfc / .zip |
| NDS | DraStic | .nds / .zip |
| GBA | RetroArch (mGBA) | .gba / .zip |

以下平台**有 ROM 才显示**：

| 平台 | 模拟器 |
|------|--------|
| PS1 | RetroArch (PCSX ReARMed) |
| PSP | PPSSPP |
| Sega Saturn | RetroArch (Yabause / Kronos) |
| Wii | Dolphin |
| GameCube（NGC） | Dolphin |
| Sega Genesis / MD | RetroArch (Genesis Plus GX) |
| Game Boy / GBC | RetroArch (Gambatte) |
| N64 | RetroArch (Mupen64Plus) |
| 以及更多... | |

> 💡 首次运行某个模拟器时，系统会提示你下载或安装对应的模拟器 App，请按提示操作。

### 添加 ROM

1. 将 ROM 文件通过以下任一方式传输到手机：
   - **数据线 + 电脑**：直接拖拽到 ROM 目录
   - **TF 卡**：将 ROM 文件复制到 TF 卡对应文件夹，插入手机后扫描
   - **局域网传输**：使用文件管理器的 WiFi 传输功能
   - **网盘下载**：直接下载到 ROM 目录

2. 将 ROM 放入对应平台文件夹（如 GBA 游戏放入 `/roms/gba/`）

3. 在 Pantas Launcher 中执行[扫描刷新](#扫描与刷新游戏)

> ⚠️ 请确保 ROM 文件格式受支持（见附录），压缩包格式建议使用 `.zip`，部分模拟器支持 `.7z`。

### 扫描与刷新游戏

添加新 ROM 后，需要手动触发扫描让游戏出现在游戏库中：

- **手柄**：在模拟器主界面按 **X 键**
- **触屏**：点击右上角刷新图标
- **菜单**：Select → 扫描游戏

扫描完成后，新游戏会自动出现在对应平台的列表中。

> 💡 Pantas Launcher 只扫描已知 ROM 格式的文件，不会扫描视频、压缩包内容等，速度较快。

### 游戏封面抓取（Scraper）

Scraper 功能可以自动从网络下载游戏封面图、截图、简介等元数据，让游戏库看起来更美观。

**使用方法：**

1. 进入 **掌机助手 → Scraper**（或游戏库设置中找到 Scraper 选项）
2. 选择抓取范围：
   - **全部游戏**：抓取所有尚未有封面的游戏
   - **单个游戏**：长按游戏 → 游戏信息 → 抓取封面
3. 确认开始后，Scraper 在后台运行，不影响游戏

**Scraper 设置项：**

| 设置 | 说明 |
|------|------|
| 图片类型 | 封面图 / 截图 / 3D 盒图 |
| 图片分辨率 | 节省空间选低分辨率，高清显示选原图 |
| 覆盖已有封面 | 关闭可跳过已有封面的游戏，加快速度 |
| 仅 WiFi 抓取 | 建议开启，避免消耗移动流量 |

> 💡 封面图存储在 `/Pantas/scraped/` 目录，可随时删除以释放空间。

### 游戏详情与操作

**长按任意游戏**（或按 Select）弹出游戏选项菜单，包含：

- **启动游戏**：选择模拟器启动（如有多个可选模拟器）
- **游戏信息**：查看简介、开发商、年份等元数据
- **手动抓取封面**：重新从网络抓取该游戏封面
- **编辑游戏信息**：手动修改游戏名称、封面路径等
- **添加 / 移除收藏**
- **从游戏库移除**：从列表中隐藏（不删除 ROM 文件）

### 收藏与筛选

- **收藏**：按 **X 键**（或长按 → 收藏），收藏的游戏出现在"收藏"分类
- **筛选**：按 **Y 键**，可按字母、游戏时长、最近游玩、评分等排序筛选
- **搜索**：按 **Start 键**，输入游戏名称关键词

## 串流游戏

Pantas Launcher 内置串流游戏支持，可将 PC 或游戏主机的画面串流到手机，配合 Pantas Retro Pad 手柄流畅游玩。

**支持的串流平台：**

| 平台 | 说明 |
|------|------|
| Moonlight（本地串流） | 串流 NVIDIA GeForce Experience 或 Sunshine 主机 |
| Chiaki（PS4/PS5 串流） | 串流 PlayStation 主机 |
| Xbox 云游戏 | 通过浏览器或专属入口访问 |
| 其他第三方串流 | 可通过添加自定义应用入口接入 |

**Moonlight 串流快速设置：**

1. 确保手机和电脑在**同一 WiFi 网络**下
2. 在电脑上安装并运行 **NVIDIA GeForce Experience** 或 **Sunshine**，开启 GameStream
3. 在 Pantas Launcher → 串流游戏 → Moonlight，等待自动发现主机
4. 点击主机 → 输入电脑显示的 4 位 PIN 码完成配对
5. 配对后即可选择并启动游戏

> 💡 建议使用 5GHz WiFi 以获得最低延迟体验。有线网络（手机 USB 网卡）效果更佳。

## 云游戏

云游戏功能提供第三方云游戏服务的统一入口，无需安装额外 App 即可直接访问。具体可用服务因地区而异。

> 此功能需要稳定的网络连接，建议使用 WiFi 并确保带宽在 20Mbps 以上。

## 掌机助手

掌机助手是 Pantas Launcher 内置的手柄配置中心，功能相当于独立的 **GameMacro** 应用，可对连接的 Pantas 手柄进行全面配置，无需另外安装 GameMacro。

**进入方式：**
- 顶部导航栏按 **L1 / R1** 切换到「掌机助手」
- 或在任意界面长按 **Logo 键** 呼出 Pantas Launcher 后选择「掌机助手」

### 连接手柄

**有线连接（Pantas Retro Pad）：**

将手机插入 Pantas Retro Pad 的活动 Type-C 口，手柄自动识别，掌机助手顶部显示手柄型号、当前固件版本及模式。

**蓝牙连接（凯智方案蓝牙手柄）：**

1. 将蓝牙手柄切换至配对模式
2. 掌机助手 → 连接设备 → 扫描蓝牙设备
3. 选择对应设备名称完成配对

连接成功后，掌机助手会实时读取以下信息：

| 状态项 | 说明 |
|--------|------|
| 设备型号 | 用于显示对应手柄图片 |
| 固件版本 | 当前已安装固件版本号 |
| 当前模式 | Xinput / Dinput / DS4 |
| 电量（蓝牙手柄） | 当前电量百分比 |
| 充电状态（蓝牙手柄） | 是否正在充电 |

**物理按键测试：**

掌机助手 → 按键测试，按下任意按键可验证手柄各键是否正常响应。

### 按键配置

Pantas Launcher 掌机助手支持对所有手柄按键进行重映射，例如将 A/B 互换，或将背键 M1/M2 映射为任意功能键。

**重映射步骤：**

1. 掌机助手 → 按键配置
2. 选择需要修改的物理按键
3. 从列表中选择目标输出键值
4. 点击保存，手柄立即生效

**ABXY 键位布局切换（Xbox ↔ PS 布局）：**

部分游戏使用 PS 布局（A/B 互换、X/Y 互换），可在按键配置中一键切换整体布局，无需逐个手动修改。

> 💡 所有按键配置保存在手柄内部，拔出重插后配置依然有效。恢复出厂设置会清除所有自定义配置。

### 摇杆设置

掌机助手 → 摇杆设置，提供以下配置项：

**死区设置：**

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 前置死区（中心死区） | 摇杆中心的无感应范围，防止摇杆漂移 | 3% |
| 后置死区（外圈死区） | 摇杆外圈的最大输出触发范围 | 0%（无死区） |

**输出曲线：**

选择摇杆输出曲线类型，影响摇杆从中心到边缘的响应速度：

| 曲线类型 | 适用场景 |
|----------|----------|
| 线性（默认） | 适合大多数游戏 |
| 急加速 | 适合射击游戏（精准快速响应） |
| 缓入 | 适合赛车游戏（细腻操控） |
| 自定义（两点调节） | 手动设置曲线中两个控制点 |

**其他摇杆选项：**

- **XY 轴反转**：分别对左/右摇杆的 X 轴或 Y 轴进行翻转
- **左右摇杆对调**：将两侧摇杆的输出互换（适合左撇子玩家）

**摇杆校准：**

如果摇杆存在漂移或中心偏移，需要进行校准：

1. 掌机助手 → 摇杆校准
2. 不要触碰摇杆，点击「开始校准」
3. 按提示将两个摇杆各旋转两圈（完整圆形），再分别按压扳机两次
4. 按 **B 键** 退出校准模式，校准完成

> ⚠️ 校准信息保存在手柄内部，**恢复出厂设置不会清除校准数据**，请放心使用。

### 扳机设置

掌机助手 → 扳机设置，提供以下配置项：

| 设置项 | 说明 |
|--------|------|
| 前置死区 | 扳机轻微按下时的无感应范围，防止误触 |
| 后置死区 | 扳机接近全按时的触发范围 |
| 快速扳机 | 开启后，轻按扳机即可触发满值输出（适合射击游戏） |
| 输出曲线 | 类似摇杆曲线，控制扳机响应速度 |
| 左右扳机对调 | 互换两侧扳机输出 |

### 连发（Turbo）设置

掌机助手 → 连发设置，可对任意按键设置连发功能：

| 功能 | 操作 | 说明 |
|------|------|------|
| 开启连发 | 选择目标按键 → 开启连发 | 按住该键即持续连发，松开停止 |
| 开启自动连发 | 在连发状态下开启「自动」 | 单按该键即开始持续连发，再按关闭 |
| 调整连发频率 | 选择频率档位 | 低档 7Hz / 中档 15Hz（默认）/ 高档 20Hz |
| 关闭连发 | 取消勾选对应按键 | 取消成功 |

**支持连发的按键：** A / B / X / Y / LB / RB / LT / RT / L3 / R3

> ⚠️ 最多同时支持 **5 个按键**连发。超过限制时，最早设置的连发会被自动清除。

### 马达震动设置

掌机助手 → 马达设置：

- **震动强度**：0% / 25% / 50%（默认）/ 75% / 100%
- **单独调节左右马达**：左马达（通常对应低频震动）和右马达（高频震动）可独立设置

> 💡 游戏实际震动效果取决于游戏本身是否支持震动反馈输出。

### 灯光设置

掌机助手 → 灯光设置（仅适用于配备 RGB 灯的手柄型号）：

| 设置项 | 说明 |
|--------|------|
| 状态指示灯 | 显示当前手柄连接状态及模式颜色 |
| 左摇杆光圈 (L1–L4) | 左摇杆周围 4 颗 RGB 灯效果 |
| 右摇杆光圈 (L5–L8) | 右摇杆周围 4 颗 RGB 灯效果 |
| 灯效类型 | 常亮 / 呼吸 / 跑马灯 / 随音乐律动 |
| 亮度调节 | 0–100% |
| 颜色选择 | 全色域 RGB 颜色选择器 |

### 模式切换

手柄支持三种工作模式，可在掌机助手中一键切换，也可通过手柄实体按键切换：

| 模式 | 设备名称 | 指示灯 | 适用场景 |
|------|----------|--------|----------|
| Xinput（默认） | Xbox 360 Controller | 深蓝色 | 大多数 Android 游戏、RetroArch |
| Dinput | Gamepad | 红色 | 部分旧版游戏、需手动映射 |
| DS4 模式 | Wireless Controller | 浅蓝色 | PS 系列串流、需 DS4 识别的游戏 |

**实体按键切换**：`Select` + `Start` 长按 3 秒，在 Xinput 和 Dinput 间切换。

### 固件升级

定期更新手柄固件可以修复问题并获得新功能。

**通过掌机助手升级（推荐）：**

1. 确保手机连接 WiFi
2. 掌机助手 → 固件升级
3. 若有可用更新，点击「下载并安装」
4. 升级过程约 1–3 分钟，**请勿断开手柄**

> ⚠️ 固件升级期间请不要拔出手柄，也不要让手机进入息屏状态，以免升级中断导致手柄异常。

### 恢复出厂设置

> ⚠️ 此操作会清除所有按键自定义配置（包括连发、重映射、灯光等），**校准数据保留**。

**通过掌机助手：**

掌机助手 → 恢复出厂设置 → 确认

**通过手柄实体按键：**

开机状态下，同时长按 `Logo` + `Back` + `Start` 键 **5 秒**。

## 常见问题解答（FAQ）

**Q：插入手柄后 Pantas Launcher 没有识别到手柄？**

A：请检查：
1. 手机是否支持 USB OTG（绝大多数 Android 11+ 手机均支持）
2. 手柄是否正确插入活动 Type-C 口（而非充电口）
3. 尝试拔出重插
4. 前往掌机助手 → 按键测试 查看是否有输入响应

---

**Q：游戏库显示的游戏列表是空的？**

A：请确认：
1. ROM 文件放置在正确的平台文件夹中（如 GBA 的 ROM 在 `/roms/gba/`）
2. ROM 格式受支持（见附录）
3. 已在游戏库界面执行「扫描刷新」（按 X 键或刷新图标）
4. 存储权限已正确授予

---

**Q：游戏没有封面图，显示为默认图标？**

A：使用 Scraper 功能自动下载封面，详见[游戏封面抓取](#游戏封面抓取scraper)章节。少数冷门游戏数据库中可能没有封面，可手动导入图片。

---

**Q：模拟器无法启动某款游戏？**

A：请确认：
1. 对应模拟器 App 已安装（首次运行会提示安装）
2. ROM 文件未损坏（可尝试重新获取）
3. 部分模拟器（如 AetherSX2）需要从正版主机提取的 BIOS 文件，请自行了解相关法律规定

---

**Q：摇杆存在漂移（自动偏移）？**

A：执行摇杆校准即可解决，详见[摇杆校准](#摇杆设置)章节。

---

**Q：如何让 Pantas Launcher 在游戏结束后自动弹出？**

A：在 Pantas Launcher 设置 → 通用 → 开启「游戏退出后自动返回前端」。确保 Pantas Launcher 有「在其他应用上方显示」权限。

---

**Q：换了手机，如何迁移游戏库数据（收藏、游戏时长等）？**

A：
1. 登录个人中心 → 同步设置（如已开启云同步）
2. 或手动将 `/Pantas/` 目录整体复制到新手机的相同路径

---

**Q：主题商店打不开？**

A：请检查网络连接，确保 Pantas Launcher 有网络权限。如长时间无法访问，可关注官方社群获取最新状态通知。
